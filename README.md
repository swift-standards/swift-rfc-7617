# swift-rfc-7617

Swift implementation of [RFC 7617: The 'Basic' HTTP Authentication Scheme](https://www.rfc-editor.org/rfc/rfc7617.html)

## Installation

### Swift Package Manager

Add this to your Package.swift:

```swift
dependencies: [
    .package(url: "https://github.com/swift-web-standards/swift-rfc-7617.git", branch: "main")
]
```
